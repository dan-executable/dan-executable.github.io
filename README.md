# Hyeon Ho Lee - Personal Website

This website is a personal portfolio built with HTML, CSS, and JavaScript. It includes information about my education, skills, experience, and visualizations.

## Files
- `index.html` - Homepage with personal info, education, skills, and experience.
- `visualizations.html` - Page containing an SVG visualization and creative SVG art.
- `style.css` - Styling for the website.
- `main.js` - Basic JavaScript interactivity.
- `vis.js` - JavaScript code for SVG visualizations.
- `assignment3.html` - Assignment 3 page featuring four data visualizations and written reflections
- `README.md` - Description of the website and navigation.

## Usage
Open `index.html` in a browser to view the homepage. Navigate to the Visualizations page and Assignment 3 page to see the SVG drawings and video game sales data visualizations.

## Technologies Used
- HTML
- CSS
- JavaScript
- SVG for visualizations
- Vega-Lite in Observation for visualizations
